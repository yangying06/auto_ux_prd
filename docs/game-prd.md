## 一、游戏概述

### 1.1 核心定位

| 项目 | 内容 |
|------|------|
| 游戏类型 | 2D 横版动作 + 资源采集 |
| 目标平台 | Web（手机端） |
| 单局时长 | 10~20 分钟 |
| 核心循环 | 采矿 → 升级装备 → 深入地下 → 采更多矿 |

### 1.2 核心玩法描述

玩家扮演一名太空矿工，在程序生成的地下矿洞中向下挖掘。收集矿石换取金币，升级镐、背包和护甲，同时躲避或击败地下生物。目标是到达地图最底层获取传说矿石并逃出地面。

## 二、数据结构

### 2.1 玩家数据（Player）

```typescript
interface Player {
  // 基础属性
  hp: number          // 当前生命值
  maxHp: number       // 最大生命值（初始 100）

  y: number           // 世界坐标 Y（像素）
```

## 三、地图生成规则

### 3.1 分层结构

| 层 | 行范围 | 地块组成 | 特殊特征 |
|----|--------|---------|---------|
| 地表 | 0~4 | 空气 + 草地 | 出发点、商店 |
| 浅层 | 5~29 | 泥土 70%、石头 20%、煤矿 8%、铁矿 2% | 商店（行 25）|
| 中层 | 30~59 | 石头 55%、泥土 15%、铁矿 15%、金矿 10%、红宝石 5% | 商店（行 50）|
| 深层 | 60~89 | 石头 60%、金矿 15%、红宝石 15%、钻石 8%、熔岩坑 2% | 商店（行 75）|
| 底层 | 90~119 | 石头 70%、钻石 20%、传说矿 1（唯一）| Boss 房 |

### 3.2 洞穴生成

使用 Cellular Automata 算法生成自然洞穴：
1. 随机填充 45% 的格子为实心
2. 迭代 5 次平滑（周围 8 格中实心 ≥ 5 则变实心）
3. 保证玩家出生点附近 5×5 区域为空气

### 3.3 商店生成

每隔 25 行在随机 X 位置生成一个 3×4 的空腔，放置 NPC 商人实体。

## 四、核心机制

### 4.1 挖掘机制

- 玩家面向地块，长按挖掘键（鼠标左键 / Space）
- 挖掘进度条显示在地块上方
- 进度完成 → 地块变为 `air`，掉落物品弹出
- 体力每次挖掘消耗 1 点；体力归零时挖掘速度降低 50%
- 体力自动回复：1 点/秒（静止时 2 点/秒）

### 4.2 移动机制

| 操作 | 键位 |
|------|------|
| 左右移动 | A / D 或 方向键 |
| 跳跃 | W / 上方向键 / Space |
| 攀爬梯子 | W / S |
| 互动/挖掘 | 鼠标左键（点击目标格）|

物理参数：
- 重力：`980 px/s²`
- 跳跃初速：`-480 px/s`
- 最大下落速度：`600 px/s`
- 水平摩擦（地面）：`0.8`

### 4.3 战斗机制

- 玩家无主动攻击；镐挖掘时对相邻敌人造成伤害（= 镐等级 × 10）
- 敌人接触玩家造成伤害，有 0.8 秒无敌帧
- 玩家 HP ≤ 0：显示游戏结束界面，背包物品全部丢失，回到地表

### 4.4 商店交互

- 靠近 NPC 按 E 键打开商店界面
- 商品：生命药水（30 金）、各级装备、背包扩容（+4 格，100 金）
- 背包内物品可全部卖出（按物品单价计算）

### 4.5 胜利条件

挖到传说矿石后，右上角出现「返回地面」提示；玩家返回第 0 行触发胜利动画和分数结算。

## 七、技术规格

### 7.1 渲染

- 引擎：Phaser 3（或纯 Canvas 2D API）
- 分辨率：960×600 逻辑像素，自适应缩放
- 地块尺寸：32×32 像素
- 视口：跟随玩家，平滑插值（lerp factor 0.1）
- 光照：玩家周围半径 6 格可见，边缘渐变遮罩

### 7.2 存档

```typescript
interface SaveData {
  version: number      // 存档版本（当前 1）
  player: Player
  world: {
    seed: number
    tiles: Tile[][]    // 仅存储被修改的格子
  }
  playTime: number     // 累计游玩秒数
  savedAt: string      // ISO 时间戳
}
```

存储方式：`localStorage`，键名 `space-miner-save`。

### 7.3 性能目标

| 指标 | 目标 |
|------|------|
| 帧率 | 稳定 60 FPS |
| 地图加载 | < 200ms |
| 内存占用 | < 150MB |
| 首屏加载 | < 3s |

### 7.4 程序生成参数（可调）

